This project is a frontend dashboard UI created during a hackathon as part of an AI Career Advisor platform.

## Overview
The goal of this contribution was to design and implement a clean, modern dashboard interface that presents user progress, goals, and skill insights in a clear and engaging way. The focus was on layout, visual hierarchy, and rapid UI development under hackathon constraints.

## My Contribution
- Designed the dashboard layout and user flow using Canva
- Defined the structure for the hero section, sidebar navigation, and card-based components
- Used AI-assisted tools to accelerate the design-to-code process
- Refined the generated HTML and CSS to match the intended UI and improve visual consistency

## Tech Stack
- HTML5
- CSS3
- Canva (UI/UX design)
- AI-assisted tools for faster frontend implementation

## Notes
This repository represents the frontend UI layer of the project. Backend functionality and full interactivity were outside the scope of this contribution during the hackathon.
